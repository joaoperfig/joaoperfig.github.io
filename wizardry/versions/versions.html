Version History

The following is the list of game releases in reverse chronological order.



0.2523

Monday, 29th January 2018

Major changes:
   Added an AI that can actually win Games.
   Major changes to RuneScribe loadout.
   Added Necromancer.
   Added Warlock.
Minor changes:
   Changed how basic kicks work (now pushe other wizards).
   Nerfed Occultist.
   Nerfed Thaumaturge.
   Buffed Pyromancer.
Bug fixes:
   Pyromancer's big Fireball will no longer instantly disappear when cast next to bottom wall (fixed again).
   Occultist no longer disappear when teleporting to a ball held by a tentacle.
   Fixed objects not despawning.



0.2490

Wednesday, 19th January 2018

Major changes:
   Added Full gamepad support.
   Allows toggling through any combination of input methods.
   changed how Wizard physics work.
Bug fixes:
   Pyromancer's big Fireball will no longer instantly disappear when cast next to bottom wall.
   Occultist no longer disappear when swapping with Teleporting occultist.
   Occultist no longer disappear when swapping with Possessing occultist.



0.1490

Tuesday, 18th January 2018

Major changes:
   Added basic AI.
   Major visual overhaul of field.
   Added ESC button to return to menu.
Bug fixes:
   Fixed Random Wizard's colors being to dark.



0.1432

Monday, 15th January 2018

Major changes:
   First Release with installer.
   Replaced Thaumaturge skin with "Random Wizard" skin.
   changed keyboard controlls.
Bug fixes:
   Fixed bug with selected wizards changing after goals.



0.1410

Sunday, 14th January 2018

Major changes:
   First public Alpha Release.
   First Release with menu.
   Added Occultist.
   changed keyboard controlls.
Bug fixes:
   Fixed menu size.
